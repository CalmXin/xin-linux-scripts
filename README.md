## 脚本

项目特色，有意识地让所有脚本都基于 python 实现。

### 一键安装 python

```shell
curl -fsSL https://scripts.pythonlove.com/install_python.sh | bash
# 或
wget -qO- https://scripts.pythonlove.com/install_python.sh | bash
```

### Run scripts in bin/

```shell
# 安装 Docker 环境
python3 bin/install_docker.py
```
